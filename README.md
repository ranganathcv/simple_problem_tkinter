# Tkinter Calculator

## Description

This project is a simple calculator built using Python and Tkinter.
It performs basic arithmetic operations including addition, subtraction, multiplication, and division.

The calculator manages state using global variables instead of using the `eval()` function.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear button
* Error handling for division by zero

## Technologies Used

* Python
* Tkinter (Python GUI library)

## How to Run

1. Make sure Python is installed.
2. Save the file as `calculator.py`.
3. Run the program using:

```
python calculator.py
```

This project demonstrates how to:

* Use Tkinter for GUI applications
* Manage calculator state with global variables
